TrussSketch Quick Reference

All functions mapped from TrussC

Example

global angle = 0.0 def setup() { logNotice("Starting!") } def update() { angle = angle + getDeltaTime() } def draw() { clear(0.1) pushMatrix() translate(getWindowWidth() / 2.0, getWindowHeight() / 2.0) rotate(angle) setColor(1.0, 0.5, 0.2) drawRect(-50.0, -50.0, 100.0, 100.0) popMatrix() } def keyPressed(key) { logNotice("Key: " + to_string(key)) }
